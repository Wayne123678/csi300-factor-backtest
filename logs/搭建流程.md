## 第3步：因子计算（factors.py）

### 9个因子


| 因子            | 计算                                | 类别  | 数据依赖                |
| ------------- | --------------------------------- | --- | ------------------- |
| `mom_5`       | close.pct_change(5)               | 动量  | close               |
| `mom_20`      | close.pct_change(20)              | 动量  | close               |
| `mom_60`      | close.pct_change(60)              | 动量  | close               |
| `rev_1`       | -ret（日收益率取负）                      | 反转  | close               |
| `vol_20`      | ret.rolling(20).std()             | 风险  | close               |
| `turn_5`      | turnover.rolling(5).mean()        | 活跃  | turnover（腾讯源缺失）     |
| `vol_ratio_5` | volume / volume.rolling(5).mean() | 活跃  | volume              |
| `amp_5`       | amplitude.rolling(5).mean()       | 活跃  | amplitude（从OHLCV计算） |
| `size`        | ln(close × total_shares)          | 规模  | close + stock_info  |


### 设计要点

- 中间列 `ret` 在 vol_20 算完后 drop，不存表
- `DROP TABLE IF EXISTS factors`：每次重跑是全量重建，不追加重叠
- `size = ln(市值)`：不取对数则茅台(2万亿)和京东方(千亿)差20倍，会压扁其他因子
- 表主键 `(stock_code, trade_date)` 联合主键

## 第4步：因子有效性分析（factor_analysis.py）

### 分析方法

1. **IC分析**：逐月 Spearman 秩相关（因子值 vs 未来N日收益）
2. **分层回测**：每月按因子值分3组（top/mid/bottom），算各组实际未来收益
3. **双去极值**：分位数 clip(1%,99%) vs MAD ±5σ，对比稳健性
4. **因子相关性矩阵**：Pearson相关，|r|>0.7标记警告
5. **IC衰减曲线**：5/10/20/40/60日展望周期，确定最佳调仓频率

### 输出三张表 + 两个附


| 表   | 内容                                               |
| --- | ------------------------------------------------ |
| 表1  | 分位数方法：IC/ICIR/P值/Top-Mid-Bot/spread/smart/胜率/单调性 |
| 表2  | MAD方法：同上结构                                       |
| 表3  | 对比表：ICIR差异/spread差异/单调性/综合评级                     |
| 附1  | 因子相关性矩阵（双方法）                                     |
| 附2  | IC衰减曲线（合成因子 × 多展望周期）                             |


### 关键指标详解


| 指标           | 含义                                | 判断标准                |
| ------------ | --------------------------------- | ------------------- |
| IC均值         | 因子值与未来收益的秩相关均值                    | 远离0                 |
| ICIR         | IC均值 ÷ IC标准差                      | >0.3可用，>0.5好        |
| P中位          | 各期Spearman P值中位数                  | <0.05理想             |
| P<0.05       | 统计显著月份占比                          | 越高越好                |
| smart_spread | IC>0时=Top-Bottom，IC<0时=Bottom-Top | 正=方向正确且赚钱           |
| 胜率           | smart_spread>0的月份占比               | >60%较好              |
| 单调性          | 三组是否按预期方向递减                       | [++]严格 [~~]弱 [XX]相反 |
| ΔICIR        | |ICIR_MAD| − |ICIR_分位数|           | 看两种方法的稳健差异          |


### 综合评级

```
[A] = ICIR>0.3 + 胜率>60% + smart为正
[B] = ICIR>0.15 + 胜率>50%
[C] = ICIR>0.1
[--] = 弱或无效
```

| v   | 改进                   | 原因              |

| v1  | 基础IC+分层回测            | 初始实现            |

## 第5步：多因子合成（factor_synthesis.py）

### 方法

Z-score标准化 + 三种加权 → 综合得分。


| 方法     | 公式                                                | 特点      |
| ------ | ------------------------------------------------- | ------- |
| 等权     | score = z1 + z2 + z3                              | 不看历史表现  |
| ICIR加权 | score = Σ(w_i × z_i)，w_i = |ICIR_i|/Σ|ICIR|       | 信任稳定因子  |
| IC均值加权 | score = Σ(w_i × z_i)，w_i = |IC_mean_i|/Σ|IC_mean| | 信任高IC因子 |


### 15只结论

合成未打败单因子mom_20（ICIR 0.463 vs 0.414）。原因：vol_20与mom_20相关0.43，size ICIR只有mom_20的74%。小样本下保持简单优于强行合成。

## 第8步：PE/PB 历史数据获取（2026-06-17 上午）

### 核心突破：绕过 akshare 限流

akshare 的 `stock_value_em()` 调用 ~12 次后触发东方财富限流。
**解决方案**：阅读 akshare 源码找到真实 API 端点 `RPT_VALUEANALYSIS_DET`，直接 `requests` 调用。


| 尝试    | 方法                               | 结果                 |
| ----- | -------------------------------- | ------------------ |
| 1     | `ak.stock_value_em()` 逐只         | ❌ 12次后限流           |
| 2     | 百度源 `stock_zh_valuation_baidu()` | ❌ 单次30s+           |
| 3     | `efinance` 库                     | ❌ 底层调 push2his，同被封 |
| **4** | **直连东方财富数据中心API**                | ✅ 300只全量，568,520条  |


### 技术细节

```
URL: https://datacenter-web.eastmoney.com/api/data/v1/get
报表: RPT_VALUEANALYSIS_DET
字段: SECURITY_CODE, TRADE_DATE, PE_TTM, PE_LAR, PB_MRQ, ...

```

`build_valuation.py`：直连API，0.5s间隔 + 每15只降温30s → 约9分钟完成300只。

### 列补全：turnover

turnover（换手率）= volume ÷ total_shares，交易所标准定义。
`build_db.py` 第2步从 OHLCV + stock_info 自动计算，100%覆盖。

amount（成交额）不可达：所有Python库底层调 `push2his.eastmoney.com`，该域名对中国大陆IP拒绝连接。不影响任何因子。

## 第9步：CSI 300 因子分析结果（2026-06-17）

### 9个价格因子：全线失效


| 因子     | ICIR   | smart  | 评级  | 15只→300只变化        |
| ------ | ------ | ------ | --- | ----------------- |
| size   | -0.262 | +2.39% | [C] | 方向反转(+0.34→-0.26) |
| vol_20 | -0.181 | -1.52% | [C] | 方向反转              |
| amp_5  | -0.165 | -1.44% | [C] | —                 |
| mom_20 | -0.080 | +0.33% | [—] | 从+0.46崩塌          |
| 其余5个   | <0.09  | —      | [—] | —                 |


### 核心发现

1. **样本选择偏差威力巨大。** 15只精选蓝筹中mom_20 ICIR=0.46，300只CSI 300崩塌为-0.08。
2. **P值显著 ≠ 经济显著。** 65-82%月份P<0.05，但IC均值仅0.02~0.05。
3. **2024-2025 A股是大盘价值行情。** size方向反转，国家队托市主导。

### PE/PB因子结果（2026-06-17 上午）

加入PE/PB后11因子全量分析。结论不变：**全线失效。**


| 因子     | ICIR(MAD) | smart  | 胜率  | 评级  |
| ------ | --------- | ------ | --- | --- |
| pe     | -0.266    | -0.60% | 53% | [C] |
| size   | -0.262    | +2.39% | 35% | [C] |
| pb     | -0.210    | -1.06% | 53% | [C] |
| vol_20 | -0.181    | -1.52% | 47% | [C] |
| amp_5  | -0.165    | -1.44% | 47% | [C] |
| 其余6个   | <0.1      | —      | —   | [—] |


**评级修正**：修复了[B]评级不检查smart方向的bug。PE和PB的smart为负，正确降级为[C]。

**衰减修正**：修复了低IC仍判"滞后生效"的bug。增加阈值 |IC|<0.10→"平坦(无效)"。

### 最终结论

项目从15只精选→300只全市场完整验证：**2024-2025年A股不适合传统因子策略。**

- 国家队托市压平了价格发现机制
- 大盘价值风格主导（size因子方向反转）
- 样本选择偏差威力巨大（15只→300只，结论完全颠倒）
